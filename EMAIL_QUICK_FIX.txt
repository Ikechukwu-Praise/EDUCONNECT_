## If Email Still Fails

### Option A: Check Spam Folder
- Email might be in spam/junk
- That's normal, add domain to safe senders

### Option B: Use Gmail SMTP (More Reliable)
1. Go to: https://myaccount.google.com/security
2. Create "App Password" for Mail
3. Go to Supabase ‚Üí Authentication
4. Find "SMTP Settings"
5. Fill in:
   - Host: smtp.gmail.com
   - Port: 587
   - User: your-email@gmail.com
   - Password: (16-char app password)
6. Save and test

### Option C: Skip Email for Now (Dev Only)
1. Authentication ‚Üí Settings
2. "Require email confirmation" ‚Üí OFF
3. Users instant access
4. Enable it later for production
